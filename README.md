豊富さについては、前掲表312でこの三〇年間にどう変化しつつあるかを示してある（47ページ参照）。ここでは、別の形で問題にしてみることにしより。
豊富さの中身は、広がりと深さである。広がりはWidthのことである。この広がりは売価の種類の多いことでないことは、すでに明階にした。では、いったい何が揃っていれば広がりといえるのだろうか。その内容は二つに分かれる。
第一は、時代的変化で中心テーマが違うという点だ。それを示したのが、前掲の表3ー2なのである。しかし、これは過渡期のものだ。競争状態が起こったときに問題になるものは、二つめのテーマである。
その二つめとは、選びやすいかどうかという問題だ。選びやすさとはなんのことか。それは客が探しはじめたときに、あっという間に二品日か三品目を選び出せることである。二、三品目を拾い出したあと、どれも最良の値打ちにみえるので、そのうちの一つを選ぶのに思い惑うことを、比較購買の楽しさと表現するのである。
しかし、日本の実情は全く逆が多い。最初の二、三品目を選び出すことに非常に時間がかかる。そのとこちで客は探したり、待ったり、迷ったりするのである。これは選ぶ楽しさではなく、探す苦痛である。そのくせ、せっかく探しえたものは一品目にすぎない。これでは専制政治のもとで、配給制をしかれているのと同じで、比較講買の楽しさはなに一つ残らないのだ。店が客に苦痛を与えるか、楽しさを与えるのかを分けるのが、この広がりという問題なのである。
したがって、ここでいう広がりとは、部門や、品種や、エニットや、品目アイテムや、単品（SKU）の多いことでは決してない。陳列量が多いことだとも、単純にいいきれないのである。客の求めている物が集中的に存在することなのだ。
たとえば背広を買うことにしよう。一万二、〇〇〇円から三〇万円までの紳士服がいかに揃っていたとしても、あなたにとって魅力あるものではない。なぜならば、あなたは最初から三万五、〇〇〇円前後、あるいは五万円前後を自分にとっての値ごろと想定しているはずだ。したがって、他の価格のものはいくらあっても関係がないのである。しかも、あなたの体型に合ったサイズでなければ困るのだ。
だとすれば、あっという間にあなたの目ざしていた値ごろのあたりで、あなたの体型に合ったサイズのものが何着か見つかることが必要なのである。日本の現状は、この第一段階で延々と時間がかかりいらいらしてしまう。そのあとであなたは、どのスタイル、柄にするかを、時間をかけて検討すればよい。それがショッピングの楽しさなのである。婦人衣料については、この面ではまるきりだめな店が大部分であることを自覚すべきである。食品でも同じである。
しかし、もう一歩進めたのがチェーンストア・システムにおける広がりである。それは品種の数がふえることである。それも、使うときのコーディネーション（Coordination）を前提にするのである。同時に使うものは服飾でも、住まいでも、食べ物でも、すべてそこで間に合わなければならないのである。
