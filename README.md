小売業でもフード・サービス業でも、共通に扱うべき商品については五つの大原即がある。そのテーマは、の豊富さ、⑧安さ、③信頼性、@便利さ、@古くないこと、の五項目だ。
こういえば、しごく当たり前に見えるが、その一つ一つを検討してみると、いずれの項目も、そう簡単には実現できない技術チーマであることがわかるだろう。以下。そのための法則を顔討するわけだが、詳細は無著「商品構成」（「チェーンストアの実務原則シリーズ」実務教育出版発行）をぜひご覧いただきたい。本書はその競編として書かれているものである。
なお、商品総成（証）とは、品目<証18）と陳列量、あるいはメニュー品目とその組合せ方という意味であり、品目という言葉には商品の性質、機能、用途が含まれていると、受けとめていただきたい。
愛高さの問題で第一に大切なことは、十年ごとに意味が変わってきているということだ。流通の世界では、だれもが豊富な品揃えが必要だという。
しかし、豊富さとは何かの数量が多いことを意味しているはずなので、私がそれは何の数のことかと質問しても、即答できる人はほとんどいない。つまり実務上これほどあいまいな表現はない言葉なのである。たとえば、一九六〇年代にまず商品構成上やるべき課題は、部門数をふやすことだと主張した。しかし一九七〇年代に入ると、ふやすべきは品種数であり、一九八
〇年代は品目数と変化させている。二〇〇〇年代は、実用品の中でコーディネートする品種と品目数となる。逆に部門数は、単純にふやしてはいけなくなる。部門間で客の購買頻度が同じで、さらに適正規模の面積が確保できる場合のみ部門数をふやしていいこととしている。言い換えればライン・ロビング<註21>であり、ワン・ストップ・ショッピングへ註22>を否定しているわけだ。
現時点では、このほかふやすべきものは、表1ー2の日に記されたもので、逆に的に列記したものは、減らしていく努力が正当なのである。それなのに実際には何の項目をふやし、国の項目は減らしていないだろうか。
「豊富さ」という言葉をめぐる商品構成の原則は、時流とともにどんどん変化していくものと理解してほしい。
さらに同に列記したものは、ふやしてもいけないし、減らしてもダメというものである。そして最後の仕上げが、@に記した深さ（Depth）と広がり（Width）の二つをも実現することである（次表113参照）。
商品部も店舗現場も、この表1ー2と全く逆行していても平気でいられるのは、日本とアメリカとの店舗現場のストア・コンパリゾン（独ページ参照）不足によるものである。
品揃えの深さとは、ある部分の商品が極端に多いことをいう。ある部分というのは品目数の場合もあり、陳列量の場合もある。Aという品種については一〇〇品目あるが、他の大部分の品種については五～二〇品目である場合、品揃えはAについて深いというのである。あるいは同じ品種の中でも、Bという品目だけがフェイシング＜注23>数を二〇～三〇もとっているが、しかし他の大部分の品目はフェイシング数が一～六にすぎないというときも、深いというのである。
この「深さ」は江戸時代から商法の極意とされてきたのだが、最近の店はこの深さがなくなってきたために、店にエキサイトメント（Excitement ＝興奮）がなくなっていると批評されるのである。もっと別の言葉でいえば、重点がないのだ。
いまコア（Core）づくりというのが、アメリカのチェーンストア業界では競争対策としていちばん重要な商品政策になっている。コアというのは核ということで、つまりコアづくりとは戦いの武器の主力になる商品づくりということなのだが、それはここでいう深さ（Depth）のある品種および品目揃えをするという意味なのである。
君たちの売場には何もかも万遍なくある。そのかわり陳列量（メニュー数）は、いずれも同数に近い。スーパーマーケットにいたっては、フェイシング数が多くて四、少ないものは一しかないという場合さえある。たまに多いと思っても、フェイシング数はせめて六。生鮮食品売場などは、もっとひどい状態である。
深さ（Depth）が、まず来店客を興奮させるのだ。これは、食堂のメニューでも同様だ。フード・サービス業の場合は実に簡単にわかる。
この深さの強調こそ、差別化（Specialization）なのである。もう一つは、Width ＝広さである。広さというのは、多品目という意味ではない。
使う立場に立ってみて、便利に揃っているかどうかということである。つくる立場、売る立場、メーカーや問屋の立場で揃っているかどうかではない。同じときに同じ場で使うものが揃っているのが、第一段階のコーディネーションである。鍋料理をするとき、動物蛋白の種類ごとによくあう野菜は揃っているか。
服飾ならば上から下まで、頭から足先まで揃っていなければ、そのルックや用途を売ったことにならない。それなのに、セーター、ブラウス、スカート、スラックスさえあれば揃えたつもりになっている。いったい、それだけの品種で一人の人間のコーディネーションができるのかということだ。〇〇〇円のスカートをはいたり、いわんや牛革のハンドバッグを持っているのに、桃は二、八〇〇円のを履いたりしている。これではチグハグで、モードもルックもテイストも全く調和がない。それ以前の問題だ。
コーディネーションの第二段階は、配色の調和である。これは、特に住まい用品と服飾用品で問題だ。フロア・カバリング、ウォール・カバリング、ウインドウ・カバリングが同じ主調色で配色できるように売られているか。ルックもモードもテイストも同一になるのか。それでいて、材質もいろいろあるのだろうか。これらが、当たり前の品揃えとして解決されているのがアメリカのチェーンストアなのだ。
少なくとも平成十年まで日本では、どこのチェーンでも、その社のストア・ブランドへ28>だけで住まいづくりをしても、配色とデザインがチグハグで、そこに住む人は落ち着いた暮らしはできなかった。そういった見方で、もう一度、自分が扱っている分野は深さがあるか、広さがあるのか、と見直す必要がある。もちろん、それら商品は、その用途に見合う性質を十分に保有していなければならない。仕入れ行動に入る前に、このことも確立しておかねばならぬ原則である。
安い売価は、だれが考えても望ましいものであるはずだ。ところがわが国では、「安かろう悪かろう」という迷信が依然として存在し、特に「専門店」志向のお店にその傾向が強い。そこで、安さという言葉の意味を明確にしておくことが先決なのだ。①同じ銘柄、品番、品目の場合、②品質は同じという場合、③名称だけ同じ場合、の三通りに分かれるが、③は明らかに品質が違う以上、後でいう第二の安さに含まれるだろう。
右の①、②はふつう「•割引」と販促されるもので、アメリカのチェーンストア業界ではチープ・プライス（Cheap Price）<註29>と名づけられている。流通段階で、ふつうの業界慣習とは異なる取引方法ないし条件で集荷されたものである。しかし中には、かけ値の小さいベンダー（取引先）を探し出すという労力的努力だけで実現される例もある。仕入れ担当として、まずは取り組むべき当然の努力である。日本ではこのチープを忌避すべきことのように教える学者もいるが、“じ品なら必ず安い”ことは、常に消費者にとって望ましい条件である。
第二の安さは、ディスカウント・プライス（Discount Price）＜註30＞だ。これも日本では割引値のことと混同しやすいが、本来の米語の意味は、生産（加工）段階でこれまでとは違った方法でつくられた結果、安くなったときの価格である。その点、チープとは対象も対策もまるで異なるのである。
仕入れ方としてこの問題と取り組むには、チープ商品の場合と比べてずっと長い研修期間を必要とする。アメリカのチェーンストア経営では、チープ・プライス商品の仕入れが十分にできるようになったあと、次の能力段階として、このディスカワント・プライス商品に取り組ませる仕組みになっている。
視点を変えてみると、チープ・プライスは仕入れルートと取引条件への挑戦、デイスカウント・プライスは商品開発への挑戦、と区別して表現することもできる。
第三に安さとは、客が払いやすい売価という性格ももっている。
どんなにすばらしい効用をもった商品でも、それが多数の消費者に受け入れられやすい売価でなければ買われない。
これを古来、「お手ごろな値段」と日本では表現してきた。それは気軽に払える売価、あるいは納得して払える売価でもある。アメリカでは、ポピュラー・プライス（Popular Price）<註31＞と表現する。こうして三通りに大別した内容を、一言で廉価＝安い売価というのである。この康価は、「繁盛」と「成長」と「貢献」という企業業績と技術水準の発展史の中で、どの段階でも確保されねばならない命題である。
