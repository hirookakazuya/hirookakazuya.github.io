豊富さのもう一つの大条件は品揃えの深さである。深さとは、Depthのことだ。
これはチェーンストア・システムでは、商品の強さを示しているものとされている。さきの広がりは便利さであり、ここでいう深さは強さである。
それには三つの原則がある。
①ある品種のみ品目が多く、他のより多くの大部分の品種は、それぞれの品目数が比較的少ない状態であること。つまり、どの品種にもまんべんなく同じょうな品目数があるという状態ではないのだ。格差がついているのであ
る。る。
② 品種ごとに、特定少数品目については陳列量が過大であり、その他の大部分の品目については陳列量が相当な状態であること。これもまた格差が、陳列量について大きく開いていることを意味している。
③同時に使うものが揃っていることを、コーディネーションという。つまり、品種や品目や、陳列量について、大きな在庫格差、あるいは面積格差の、あるいはフード・サービス業ならば、メニューの行数の格差のあることが深さなのである。そして、それをつきつめていったときに、もう一度広がりの最終段階のコーディネーションが、テーマとして浮かび上がってくるのである。したがって、競合段階では広がりのある品揃えが最初の突破口であり、競争段階に入れば、深さで個性を出すのである。このあたりの原則をつきつめていくと、ひとりでに同じ業種や業態に屈していても、企来ごとに違う品揃えになっていくのである。昭和五十年代になぜ日本のそれそれのスーパーや専門店などが、他社とよく似てしまったのかという原因は、実はこうした単純明快な原理原則を無視していたものである。
こうした広がりと、深さの同時成立を実現することが、まず商品の組合せ方の基本技術ということができるだろう。
