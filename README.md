売価は商品構成、つまり品揃えの基本条件であると述べたが、ほんとうはもっと重要な意味をもっている。というのは、どのように品質や機能がよくても、客の好みにかなったとしても、それは客が気軽に払える価格でなければ販売には結びつかないという点である。
しかももっと重要なことは、店に対する客の頼は、売価をめぐるものであるという点だ。一般にはその店の伝統だとか、そこで売られている商品の品質であるとかが、客の店に対する頼感の中身だと思われがちだが、各種の調査をしてみれば、それは属性であっても本質ではないことがわかるのである。その店で売っている物なら、どのような品でも必ず買うというわけでは、決してないからである。その店を言頼しているからといって、その店の品揃えのすべてを買うわけではないか5だ。
となれば、その店に対する究極的な信頼は価格についてのものなのだ。もちろん、そのなかには決して不当な売値はついてないとか、その売価にふさわしい品質をもっているとかいうふうに、さまざまな形になるけれども、その根源にあるものはその店の提供する売価に対する頼なのである。となれば、売上高を維持するのは客の人気であり、客の人気は、客の店に対するストア・ロイヤルティ（StoreLoyalty）に左右される。その基盤が売価ということになるのだ。
では、その頼性についてチェーンストア経営の経験法則は何を教えているのであろうか。それの出発点が価格帯、努力方向が、セグメンテーションという二つのコトバなのである。価格帯（Price Range）とは、品種ごとに売価の上限と下限の間隔のことである。
セグメンテーションとは、製造業の世界では市場細分化販売政策のことをさすが、チェーンストア経営ではそうではなくて、扱い商品の価格帯の幅を狭め、わが社の売価の特徴を、お客にはっきりと打ち出すことを意味している。
商品構成グラフを描いてみよう。横軸に売価を、縦軸に陳列量をとることで、図911 （18ページ）のような形にグラフがなったとしよう。この時、価格帯とはその商品品種の売価の上限と下限との間のすべての売価をさしている。ただし図91
1は山に裾野ができている形になっている。つまり、陳列量のごく少ない売価の品もあることになる。
次に図912（13ページ）をごらんいただきたい。このグラフは山の頂きしかない。裾野がなくなっているのである。
アメリカの軍隊に「レインジャー部隊」というのがあるのをご存知だろうか。これは山岳部隊のことだ。レンジとは英語で山の稜線のことである。だから、商品構成グラフが山の稜線だけでの線がない形をしている時は、客にとってのプライス
・イメージが強烈となるのである。従ってチェーンストアとして実現したいあるべきプライス・レンジの形といえよう。
この場合、扱う以上はどの売価の商品もそれぞれ陳列量がどっさりあるという意味であり、裏返せば陳列量の少なすぎる品目は存在していないことになる。
セグメンテーション、すなわち扱い商品の価格帯（プライス・レンジ）の幅を狭めるということは、ドライ商法で売れるものだけを徹底的に品揃えするし、死に筋がない、という状態になるわけである。
ついでにいえば、価格帯の違いは、日本では所得や趣味の違い、客層側の相違による品揃えの違いとされてきた。
しかし、欧米におけるチェーンストア産業の経験法則では、それは用途の違いであり、同時に購買頻度の違いであり、またそれは潜在客数の違いと受けとられている。
つまり、一人の人間がAという価格帯の品も、Bという価格帯の品も、Cという価格帯の品も買うし、また使用する。しかし、そのときの購買動機と使用目的ないし用途は、それぞれが違っている。買物の回数も、使用頻度も、使用期間も違うという考え方なのだ。
図911よりも図912の形の商品構成グラフになっているほうが、客にとって価格イメージは明確である。そして、客の価格についての言頼度を高める方向に影響力をつくりあげるのである。
セクメンテーションの具体的な進め方の第一は、売価の上限を下げていく（商品構成グラフを左へ移動させる）ことである。ところがこの数年間、わが国における小売業やフード・サービス業は、全くそれとは逆に、商品構成グラフで右のほうへ、すなわち売価の高いほうへと、どんどん大きく広げていくことが多かった。これでは、売価についての客の倍頼度を深めることはできない。逆に、客からだんだんと見放されていったのである。
売価の上限を上げていく理由を、より豊かにとか、より文化的に、よりファッショナブルにと、さまざまに表現をしてみても、結局は客の借頼度を下げていくことにしか結びつかないのである。
売価の上限と下限の間隔を狭めるときには、上のほうへ狭めていくのではなく、下のほうへ狭めていかなければ、客数をふやし、販売量を増加させることはできない。つまり、売価の上限を下寄せしていくということになるのである。
