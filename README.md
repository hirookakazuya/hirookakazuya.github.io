パイイング活動で最も大事な急所は、調査に時間をかけることである。総実働時間の九割は調査、一割が商談と考えたい。商談は長時間の調査の締めくくりであって、パイイング活動の主体ではないのである。
ところが、現実の商品部の動き方は逆である。調査にかける時間はゼロに近く、いちばん多いのがオフィス・ワークである。しかもその実態は、昼間は電話番と外部からの訪問者への応対だけ。午後六時半すぎに落ち着いたと思ったら、会議が始まる。だから、どこにも調査に行けないという状況ではない。
こうした事務作業は、商品部のオフィス担当の人びとにさせることが原則だ。
もともとバイヤーとかマーチャンダイザーは、タレント・スペシャリストであり、ほんものなら当然に一人ごとに秘書（Secretary）がつくべきだ。このほか、さらに商品部にはオフィス・マネジャーがいて、その下に商品部専属の事務員グループがいなければならない。年季が入っていない仕入れ方には当分一人ごとに秘書はつかないにしても、オフィス・ワーク組が別にいなければならないのである。
マーチャンダイザーというのは、電話に出ない人ともいえる。アメリカではマーマーチャンダイザーが電話に出てはいけないという極論の理由は、事前に手が打たれており、電話は緊急事故処理でしかありえないと考えるからだ。
それ以外のことで電話に出ているのは後手に回っている証拠であり、電話を使わないでもやってゆける、つまり計画的に行動をしているということが、タレント・スペシャリストとしての誇りなのである。電話に出ないし、伝票も自分では書かない、印も押さない、書類を読むのはなるべく夜というのが、タレント・スペシャリストの行動形式である。もちろん、桃書は電話係である。
というのが、パイヤーの本来の姿である。商品部員の評価のものさしは、一年間の旅行距離数といってもいい。
社内情報でいちばん大事なのは、バイヤー自身がわが店で自ら確認した情報である。第二が店からきた情報であり、第三が数表である。コンピュータ時代に入ってもこの比重は変わらない。
コンピュータがどんどん打ち出してくる数表でことたれりというのは、コンピューダ・ハードウェア・メーカーのセールスマンの口車に乗せられているだけだ。ところが、この最も大事な第一の方法を知らない、だからやったことのないバイヤーが日本には多い。
まず、店舗あるいは売場現場でバイヤー自身が行なうべき客の動向調査の方法を問題にしよう。それは、試売中のバイヤーが行なうべき自身の目による「観察」と「分析」のノウハウでもある。ここでいう「観察」とは眺めるという意味ではなくて、問題点を発見することである。
わが国の慣習では、店舗あるいは売場での問題点発見は、店舗現場にいる人びとの任務とされてきた。店側で発見した観察結果を本部商品部に通報することと、それによる改善を店現場で直ちに実行することが期待されているのが実情だった。
では、現実にそうした期待がうまく結実しているのかといえば、否である。理由は、第一に熟練者（ふつうはビジネスの世界では一五年以上をいう）でないと、本当の「観察」はできないこと、第二に観察結果の報告”表現がきわめてむずかしく、伝えにくいこと、第三に観察に続く「分析（原因や事情を推定すること）」は多数の資料や情報を必要とすること、第四に自分の目による印象は強烈だが、他人の報告は重点が伝わりにくいこと、などによるものである。
つまり、バイヤーは自ら「観察」と「分析」を現場でやるべきなのだ。
しかも、この行動慣習は試売だけではなくて、商品構成を決めたあとも、その維観察の主な方法は次のように五通りがある。
観察の主な方法は次のように五通りがある。
①死に筋原因の確定
②客の質問内容（分類、売場関連、陳列線など）
③ 客が立ち止まって注視あるいはさわったときのそれぞれの原因
④ 売場構成（商品ラインごとの位置と広さ）とフェイス＜註72＞とフェイシング⑤スポッターとショーカードと変化陳列（アクセントかオープン・ディスプレイ）によるアピール度

© 品傷み（パッケージを含む）
◎陳列ミス（陳列してあるが客には見えない、さわれない位置にあること）

チェーンによっては、この死に筋原因の確定を最も重要としている。なぜなら、死に筋原因が本当に確定すれば、品目カットもベンダーのカットもできるが、それ以上に大事なことは、二度とその傾向のものは仕入れないという制限枠が確立できるからである。
