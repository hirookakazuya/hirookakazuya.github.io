まず、扱うべきもののほうから論じてみよう。表のAグループの三つはすでに説明しているので、おわかりになるであろう。このうちベーシック・アイテム（Basic
というコトバは、さきほどの適正規模論から当然に必要となってきた用語である。
必需品のすべてを扱えば、小売店として、あるいはフード・サービス業としては面積が広くなりすぎて、多すぎて経営効率があがらない。そこで、わが社独自のピックアップをすることになる。そこにはなんらかのわが社独自の方針が必要であり、その方針のもとに選び出されてわが社が扱う品が、ベーシック・アイテムである。
言い換えれば、わが社がよりベーシックだと考えることがわが社の個性であり、スペシャリゼーションになるのである。いつも売れつづける品、あるいは常に品揃えされる品という意味で、ベーシック・アイテムというコトバを使うのは間違いである。したがってベーシックは、企業ごとにその中身の違う点が特徴だと考えなければならない。
この反対語が趣味品であり、高級品であり、そして低級品である。高級品は日本では舶来品を意味することが多い。しかし、日本語でいう舶来品は高級品であり、趣味品に属するから扱うべきではないが、逆に英語の場合はこれをFrom abroad と表現する。この場合は全く意味が逆になる。なぜならば、アメリカで使う場合は、アメリカでは生産されていない有益な品を象徴し、品質はとてもよいのに、値段は国内産品よりはるかに安いと、主張をしているものと受けとられているからである。
チェーンストア産業が成立している国では、舶来品はむしろより大衆品であり、より必需品であることを示すコトバと変化するのである。これはチェーンストアとして独自の集荷ルートを開拓し、生活提案品としてその商品を提供できる技術とシステムができているからである。日本の舶来品の場合は全く逆に、プローこの高級品の反対語は大来品なのだが、ときに低級品あるいは劣質品というコトバも使われる。これは一つの実際の用途にも常に耐えられない不適切な品物、という意味である。低級品は低額品とは限らない。高額品にも低級品はたくさんあることを知っておかねばならないのである。
次に、Bグループに移ると、ホット商品とは現在どんどん売れている売れ筋商品のことである。これこそ、マス・マーチャンダイジングの対象である。短期間でいれかわりながら売れ筋となる商品がホット商品、長期間にわたり売れ筋となる商品が、クラシック商品である。
新製品のことをトレンド商品といい、トレンド商品が売れはじめてマス・マーチャンダイジング化したのが、ホット商品である。しかし、売れないためにホット商品にならないトレンド商品が多いため、トレンド商品の扱いには充分な検討を要する。トレンド商品が売れはじめたときには、いち早く取りあげねばならない。
とすると、トレンド商品を最も完全に品揃えしている店こそ、ストア・コンパリソンの対象となるのである。しかし、繰り返すが、それはホット商品にならない限りチェーンストア志向企業は扱うべきではないと考えるのである。
